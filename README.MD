# To-Do List (PHP CRUD + MySQL)

##  Sobre o projeto

Este projeto é uma aplicação simples de **lista de tarefas (To-Do List)** desenvolvida para praticar conceitos de **CRUD com PHP e MySQL**, além de **interação dinâmica com AJAX**.

O usuário pode:

* Criar tarefas
* Visualizar tarefas
* Editar tarefas
* Excluir tarefas
* Marcar tarefas como concluídas

As atualizações de progresso são feitas de forma **assíncrona com AJAX**, sem recarregar a página.

---

##  Tecnologias utilizadas

* **PHP**
* **MySQL**
* **JavaScript**
* **jQuery**
* **HTML**
* **CSS**

---

##  Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/Rafael-firmo/ToDo-List.git
```

2. Coloque a pasta do projeto dentro do diretório **htdocs** do XAMPP.

3. Crie um banco de dados no **MySQL**.

4. Importe o arquivo SQL ou crie a tabela utilizada no projeto.

5. Configure as credenciais do banco no arquivo:

```text
database/config.php
```

6. Inicie o **Apache** e o **MySQL** no XAMPP.

7. Acesse no navegador:

```text
http://localhost/ToDo-List
```
